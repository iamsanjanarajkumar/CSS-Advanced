CSS PSEUDO ELEMENTS

The css pseudo elements allow you to style the elements or parts of the elements without adding any id's or classes to them

CSS-3 introduced a new double colonsyntax for pseudo elements to distinguish b/w them and pseudo classes

selector ::pseudo element { property: value;}
